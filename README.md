Raspberry Pi smart clock with touchscreen UI, sensors, and web API.



\# Smart Clock



A Raspberry Pi–powered smart clock with a 7" touchscreen, ambient light sensor, 

temperature/humidity sensor, motion detection, and a web API for remote control.



\## Features

\- Touchscreen UI (digital, analog, dreamcore modes)

\- Auto-dimming using BH1750 ambient light sensor

\- Room climate monitoring using BME280

\- Motion wake using HC-SR501 PIR sensor

\- Countdown timer with remote API

\- Modular Python architecture



\## Hardware

\- Raspberry Pi 4

\- SunFounder 7" HDMI Touchscreen

\- BH1750 Ambient Light Sensor (I2C)

\- BME280 Temperature/Humidity Sensor (I2C)

\- HC-SR501 PIR Motion Sensor

\- Breadboard + jumper wires



\## Software

\- Python 3

\- Flask (web API)

\- smbus2, bh1750, adafruit\_bme280

\- RPi.GPIO

\- Kivy or Qt for UI (TBD)
